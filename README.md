# NBA Player Longevity Prediction
## Feature Engineering Project

### Objective
The goal of this project is to engineer features from NBA rookie statistics to predict whether a player will remain in the NBA for at least five years.

### Tasks
- Define the target variable
- Remove irrelevant columns
- Analyze feature correlations
- Engineer new features
- Handle missing values
- Produce a clean dataset ready for machine learning
